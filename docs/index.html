redline

Token and time budgeting for Claude Code.

Set a time or dollar limit. redline paces the session to finish inside it,
and stops it before it runs over.

GitHub ★ Install

 

⏱ 10:00 left · 💰 $5.00 left
it stays inside the limit you set

Why

🔴 Never over

The budget is an inviolable ceiling. Tools lock at 90 percent. New work stops at 100 percent.

🪂 Never abrupt

It wraps up and delivers a usable result. No mid-sentence kill.

⏱ Time, not just tokens

Wall-clock deadlines too. Anthropic's budget machinery has no concept of time.

☂ One umbrella

Fanned subagents share the same budget. You never budget a subagent separately.

Install

Needs the Node that ships with Claude Code. Pick one, then restart Claude Code:

brew  brew install sankalpgunturi/redline/redline && redline install
curl  curl -fsSL https://raw.githubusercontent.com/sankalpgunturi/redline/main/install.sh | bash
git   git clone https://github.com/sankalpgunturi/redline && cd redline && ./install.sh

Use

/redline 10m $5      set a budget (time + $, any combination)
/redline $2          money only      /redline 1h 200k   time + tokens
/redline 45m 10%     45 min + 10% of plan      /redline off   clear

redline pulse    all active budgets + landed-rate (live)

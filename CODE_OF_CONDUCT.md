# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

Be respectful and constructive. Harassment, discrimination, and personal attacks are not tolerated.
Report concerns to the maintainer via a private GitHub issue or the contact in [SECURITY](docs/SECURITY.md).
Maintainers may remove, edit, or reject contributions that violate this code.
